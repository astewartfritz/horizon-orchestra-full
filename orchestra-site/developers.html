API Reference
Everything you need to build
RESTful endpoints, streaming support, and full OpenAPI spec. Every endpoint returns JSON with consistent error shapes.

Authentication

API key auth via Bearer token. Supports org-level keys, user-scoped tokens, and short-lived session keys with configurable TTL.

POST /v1/auth/token

Run Task

Submit a task to an autonomous agent. Supports model selection, architecture routing, tool restrictions, and budget caps.

POST /v1/tasks

Stream

Server-Sent Events for real-time token streaming. Receive thinking, tool calls, and final output as they happen.

GET /v1/tasks/{id}/stream

Teams

Create multi-agent teams with specialised roles. Configure team size, negotiation strategy, and consensus rules.

POST /v1/teams

Fleet

Deploy and manage an agent fleet. Auto-scaling, load balancing, priority queues, and global distribution across regions.

POST /v1/fleet/deploy

Webhooks

Real-time event delivery with HMAC-SHA256 signing. At-least-once guarantees, exponential backoff, and dead-letter queues.

POST /v1/webhooks

Code Examples
From simple tasks to enterprise fleets
Copy-paste examples for common patterns. Each example is production-ready.
# Basic autonomous agent from orchestra import AgentLoop, ModelRouter, AgentConfig, create_default_tools router = ModelRouter() tools = create_default_tools(router) config = AgentConfig(model="kimi-k2.5", max_steps=20) agent = AgentLoop(router, tools, config) async for event in agent.run("Research competitive landscape for AI code editors"): print(event)
# RAG pipeline with vector search from orchestra import RAGPipeline, RAGConfig config = RAGConfig( embedding_model="text-embedding-3-large", chunk_size=512, retrieval_top_k=8, rerank=True, ) rag = RAGPipeline(config) # Index documents await rag.index_directory("/docs") # Query with citations answer = await rag.query("What is our refund policy?") print(answer.text, answer.citations)
# Parallel agent swarm from orchestra import SwarmCoordinator, SubTask coordinator = SwarmCoordinator(max_parallel=5) tasks = [ SubTask("Research competitor A pricing"), SubTask("Research competitor B features"), SubTask("Analyse market trends 2024-2025"), SubTask("Generate competitive matrix"), ] result = await coordinator.execute(tasks) for r in result.outputs: print(r.task, r.output[:100])
# Enterprise agent team with roles from orchestra.teams import OrchestraTeam, TeamConfig from orchestra.teams.pre_built_teams import research_team team = research_team( name="Market Intelligence", size=4, budget_usd=5.00, ) result = await team.run( "Comprehensive analysis of the enterprise AI platform market, " "including TAM, key players, and growth projections through 2027" ) print(result.summary) print(f"Cost: ${result.total_cost:.2f} | Agents: {result.agents_used}")

Webhooks
Real-time event delivery
Every webhook includes an HMAC-SHA256 signature for verification. Here's how to validate in your server.
import hmac, hashlib def verify_orchestra_webhook(payload: bytes, signature: str, secret: str) -> bool: """Verify X-Orchestra-Signature header.""" expected = hmac.new( secret.encode("utf-8"), payload, hashlib.sha256, ).hexdigest() return hmac.compare_digest( signature, f"sha256={expected}", ) # In your Flask / FastAPI handler: if not verify_orchestra_webhook( request.body, request.headers["X-Orchestra-Signature"], "whsec_your_secret", ): raise HTTPException(401, "Invalid signature")
import crypto from 'crypto'; function verifyOrchestraWebhook( payload: Buffer, signature: string, secret: string, ): boolean { const expected = crypto .createHmac('sha256', secret) .update(payload) .digest('hex'); return crypto.timingSafeEqual( Buffer.from(signature), Buffer.from(`sha256=${expected}`), ); } // In your Express handler: app.post('/webhook', (req, res) => { if (!verifyOrchestraWebhook( req.body, req.headers['x-orchestra-signature'], SECRET, )) return res.status(401).send('Invalid signature'); });
